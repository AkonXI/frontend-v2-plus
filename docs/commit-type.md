
feat：新功能
fix：修复bug
docs：文档改变
style：代码格式改变
refactor：某个已有功能重构
perf：性能优化
test：增加测试/测试用例
revert：撤销上一次的commit
chore：改变构建流程、或者增加依赖库、工具等
init：新建库
build：改变了build工具如grunt换成了npm
resources:静态资源
